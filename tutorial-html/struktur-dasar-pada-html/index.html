Struktur Dasar Pada Html


Macam-Macam Struktur Dasar Html

Pada html terdapat struktur yang dinamakan struktur dasar html yang terdiri dari tag doctype, html, head, dan body. Namun sebenarnya struktur dasar pada html bukan hanya itu, tetapi ada juga meta, tittle, dan link.

Contohnya saja seperti codingan dibawah ini:


<!DOCTYPE html>

<html lang="en">

<html lang="en">


<head>

<meta charset=""UTF-8">

<meta http-equiv="X-UA-Compatible" content="IE=edge">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Struktur Dasar Hatml</title>

<link rel="stylesheet" href="style.css">

</head>


<body>

</body>


</html>



Penjelasan Struktur Dasar Html

1. Deklarasi Html5

Doctype atau Document Type Decaration (DTD) adalah suatu instruksi kepada web browser bahwa code kita ditulis dengan html5.

<!DOCTYPE html>


2. Elemen

Elemen adalah suatu komponen-komponen dokumen html yang terdiri dari tag-tag atau atribut. Seperti pada codingan di bawah ini adalah elemen struktur dasar pada html.

<head>

<meta charset=""UTF-8">

<meta http-equiv="X-UA-Compatible" content="IE=edge">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Struktur Dasar Hatml</title>

<link rel="stylesheet" href="style.css">

</head>


3. Tag Pembuka dan Tag Penutup

Tag pembuka di tandai dengan <> dan tag penutup di tandai dengan </>, contoh tag pembuka dan penutup pada struktur dasar html adalah sebagai berikut.

<html lang="en"> <!-- Sebagai tag pembuka -->
Dan
</html> <!-- Sebagai tag penutup -->

Berfungsi sebagai pembungkus semua codingan kita mulai dari head sampai ke body.


<head> <!-- Sebagai tag pembuka -->
Dan
</head> <!-- Sebagai tag penutup -->

Berfungsi untuk membungkus meta tag, title, dan link pada struktur dasar html.


<title> <!-- Sebagai tag pembuka -->
Dan
</title> <!-- Sebagai tag penutup -->

Berfungsi untuk menulis judul pada web kita yang nantinya akan tampil di mesin pencarian google.
Judul yang kita tulis di tag title maka akan di tampilakan di browser seperti gambar di bawah.


<body> <!-- Sebagai tag pembuka -->
Dan
</body> <!-- Sebagai tag penutup -->

Berfungsi sebagai wadah untuk codingan-codingan kita yang nantinya code tersebut adalah perintah apa yang ingin kita tampilkan di halaman website kita nanti.


4. Tag Yang Tidak Memiliki Penutup

<!DOCTYPE html>

<meta charset="UTF-8">

<meta http-equiv="X-UA-Compatible" content="IE=edge">

<meta name="viewport" content="width=device-width, initial-scale=1.">

<link rel="stylesheet" href="style.css">

Sama seperti tag yang memiliki pembuka dan penutup, self closing tag atau tag yang tidak memiliki penutup juga ada banya sebenarnya. Sebagian kecil contohnya adalah tag <img>, tag ini berfungsi untuk menambahkan gambar ke dalam halaman website kita.



Kesimpulan

  • Struktur dasar html yang terdiri dari tag doctype, html, head, dan body, meta, tittle, dan link.
  • Doctype atau Document Type Decaration (DTD) adalah suatu instruksi kepada web browser bahwa code kita ditulis dengan html5.
  • Judul yang kita tulis di tag title maka akan di tampilakan di browser seperti gambar di bawah.
