Element Heading HTML



Element Heading

Pada HTML terdapat sebuah element untuk membuat sebuah judul dan subjudul, element tersebut dinamakan sebagai element Heading.



Jenis-jenis Element Heading

Ada 6 jenis yang terdapat pada element Heading HTML, mulai dari h1, h2, h3, h4, h5, dan h6. Apasih perbedaan dari setiap jenis-jenis dari element Heading tersebut.



Perbedaan Element Heading h1 Sampai h6

Untuk dapat melihat dengan jelas perbedaan dari element h1 sampai h6 kita perlu menuliskan kode HTML di bawah ini.

index.html

<!DOCTYPE html>

<head>

<title>Element Heading HTML</title>

</head>

<body>

<h1>Ini adalah Element Heading h1</h1>

<h2>Ini adalah Element Heading h2</h2>

<h3>Ini adalah Element Heading h3</h3>

<h4>Ini adalah Element Heading h4</h4>

<h5>Ini adalah Element Heading h5</h5>

<h6>Ini adalah Element Heading h6</h6>

</body>

</html>


Setelah selesai menulis code di atas silahkan buka hasil codingan tersebut dan lihat hasilnya.

LIHAT DEMO


Dari gambar di atas terlihat bahwa element Heading semua memiliki font tebal baik h1 sampai h6. Namun h1 memiliki kuran font lebih besar, begitu seterusnya semakin tinggi angkanya semakin kecil ukuran fontnya.


Perlu diingat bahwa element Heading hanya terdapat sampai h6 saja, untuk h7 sampai seterusnya tidak ada di HTML.
