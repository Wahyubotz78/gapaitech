Element Paragraf HTML



Paragraf Pada HTML

Pada HTML terdapat Element Paragraf, kita bisa menulis tag <p> dan ditutup dengan </p>. Sebuah paragraf selalu dimulai pada baris baru, dan biasanya berupa blok teks.


Contoh Penulisan Paragraf Pada HTML

Mari kita lihat contoh penulisan paragraf pada HTML.

Index.html

<!DOCTYPE html>

<head>

<title>Element Paragraf HTML</title>

</head>

<body>

<p>ini adalah paragraf pertama</p>

<p>Ini adalah paragraf kedua</p>

</body>

</html>


Dari codingan di atas maka akan menampilkan seperti gambar di bawah ini.

Sifat Paragraf Pada HTML

Paragraf pada html tidak akan terpengaruh ketika kita menambahkan spasi sebanyak mungkin atau melakukan baris baru pada teks yang kita buat.


Untuk lebih jelasnya bisa lihat contoh coding di bawah ini.

index.html

<!DOCTYPE html>

<head>

<title>Element Paragraf HTML</title>

</head>

<body>

<p>ini adalah paragraf 1. ini diberi spasi sebanyak mungkin</p>

<p>Ini adalah paragraf 2


ini diberi baris baru sebanyak mungkin</p>

</body>

</html>


Dapat dilihat pada code di atas ketika dalam teks kita beri spasi sebanyak mungkin dan juga baris baru sebanyak mungkin hasil dari codingan kita tidak terpengaruh pada hal itu, bisa dilihat pada gambar di bawah ini.
