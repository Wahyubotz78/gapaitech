Element Formatting HTML



Pengertian

Formatting HTML adalah tag atau element yang terdapat pada HTML untuk memberikan arti khusus pada sebuah teks.


Macam-macam Element Formatting HTML

Ada 10 Formating HTML yang sering digunakan pada HTML :

1. Untuk memberikan teks menjadi tebal bisa menggunakan code:

<b></b>


2. Sama-sama memberikan teks menjadi tebal namun strong direfresentasikan untuk teks yang dianggap penting, codenya adalah:

<strong></strong>


3. Untuk memberikan teks menjadi miring, dengan code:

<i></i>


4. Sama-sama memberikan teks menjadi miring namun “em” direfresentasikan untuk teks yang ditekankan, codenya adalah:

<em></em>


5. Untuk memberikan warna sorotan pada teks, bisa menggunakan code:

<mark></mark>


6. Untuk membuat teks menjadi kecil, codenya adalah:

<smal></smal>


7. Untuk memberikan teks menjadi seolah-olah tercoret, dengan menggunakan code:

<del></del>


8. Untuk memberi garis bawah pada teks menggunakan code:

<ins></ins>


9. Untuk memberikan Teks subskrip yang muncul setengah karakter di bawah garis normal, misallnya ketika ingin menuliskan H2O, codenya adalah:

<sub></sub>


10. Kebalikan dari sub, sup adalah Teks subskrip muncul setengah karakter di atas garis normal, misallnya ketika ingin menuliskan referensi[1] , codenya adalah:

<sup></sup>


Ini hanyalah sebagian kecil contoh elemen Formatting pada HTML untuk melihat Element Formatting secara lengkap bisa kunjungi w3schools.com.


Penulisan Code Element Formatting

Untuk dapat lebih memahami fungsi dari element Formatting HTML yang sudah dibahas di atas dapat dilihat pada code di bawah ini.

index.html

<!DOCTYPE html>

<head>

<title>Element Formatting HTML</title>

</head>

<body>

<p><b>Ini adalah Element Formating b</b></p>

<p><strong>Ini adalah Element Formating strong</strong></p>

<p><i>Ini adalah Element Formating i</i></p>

<p><em>Ini adalah Element Formating em</em></p>

<p><mark>Ini adalah Element Formating mark</mark></p>

<p><strong>BANDINGKAN</strong> ini tidak diberi element small<small> dan ini adalah Element Formating small</small></p>

<p><del>Ini adalah Element Formating del</del></p>

<p><isn>Ini adalah Element Formating isn</isn></p>

<p>Ini adalah Element Formating sub contoh: H<sub>2</sub>O</p>

<p>Ini adalah Element Formating sup contoh: referensi<sup>[1]</sup></p>

</body>

</html>
