Apa Itu Html


HTML merupakan singkatan dari Hypertext Markup Language. Yang terdiri atas tiga komponen kata, yaitu: Hypertext, Markup dan Language yaitu bahasa markup standar yang digunakan untuk membuat halaman web.


Pengertian HML

HTML adalah bahasa yang merupakan pokok utama dari hampir semua konten Web. Sebuah bahasa yang menggambarkan struktur dan isi semantik dari sebuah dokumen Web. Sama seperti yang dijelaskan dalam wikipedia yaitu, Hypertext Markup Language (HTML) adalah bahasa markah standar untuk dokumen yang dirancang untuk ditampilkan di peramban internet. Ini dapat dibantu oleh teknologi seperti Cascading Style Sheets (CSS) dan bahasa scripting seperti JavaScript dan VBScript. Saat menjelaskan tetang HTML, berarti kita tidak bisa lepaskan dari CSS dan Javascript. Pada pembuatan website kita tidak bisa memisahkan antara ketiga hal tersebut yaitu HTML, CSS dan JS.


Analogi HTML

Untuk lebih mempermudah dalam memahami apa itu HTML, mari kita buat sebuah "analogi" seperti membangun sebuah rumah?


HTML : diibaratkan seperti membangun sebuah rumah maka HTML itu adalah kerangkanya saja, hanya kerangka sebuah rumah.


CSS : Digunakan untuk mengatur mulai dari warna, ketinggian, panjang, atau lebar dari sebuah pintu, jendela, dinding dll. Intinya CSS memiliki fungsi untuk mengatur semua warna, bentuk ataupun ukuran dari panjang lebarnya sebuah rumah.


JavaScript : Akan memiliki fungsi untuk mengatur pintu rumah dibuka kedalam / keluar, mengatur saluran air, mengatur instalasi listrik seperti Ketika di Tekan ON maka lampu kamar menyala.


Contoh

Mari kita buat sebuah website yang sangat sederhana agar kita lebih memahami "analogi" yang dijelaskan di atas. Buat file bernama index.html di text editor favorit kalian, lalu tulis code di bawah ini.

index.html

<!DOCTYPE html>

<html lang="en">

<head>

<meta charset="UTF-8">

<meta http-equiv="X-UA-Compatible" content="IE=edge">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Apa Itu Html</title>

</head>

<body>

<h1>Belajar HTML</h1>

<p>Belajar Kerangka HTML</p>

<button type="button" name="button">Pesan</button>

</body>

</html>


Buka file codingan yang kalian buat tadi, Seperti yang kita bahas di atas "analoginya" ketika kita membangun rumah maka codingan HTML yang kita buat tadi adalah kerangkanya saja, yang dimaksud kerangkanya saja ialah seperti gambar di bawah ini.
