7 Extension Untuk Membantu Menulis Code DI Visual Studio CODE


Halo sobat Gapai, diartikel kali ini saya akan memberikan beberapa rekomendasi yaitu 7 Extension Yang Wajib ada Di Visual Studio Code kamu.


Extension ini sangat membantu kamu dalam menulis code pada Text Editor Visual Studio Code, baik menulis code HTML, CSS, JAVASCRIPT, PHP dan code-code lainnya.


Tanpa berlama-lama langsung saja kita ke daftar extensionnya.


7 Extension Visual Studio Code


1. Live Server

  • Extension By : Ritwick Dey
  • Downloads : +16.643.316
  • Rating: 4.4 of 5

Jika sebelumnya kalian ketika menulis code lalu ingin melihat hasil dari code kelian di web browser maka kalian harus refresh terlebih dahulu halaman web tersebut untuk melihat hasilnya. Nah jika kalian menggunakan extension ini maka kalian tidak perlu lagi refreh-refreh halaman web kalian.


2. Auto Rename Tag

  • Extension By : Jun Han
  • Downloads : +7.243.807
  • Rating: 3.4 of 5

Misalnya di code kalian ada tag h1, lalu kalian ingin mengganti tag tersebut menjadi h2, maka yang harus kalian lakukan adalah mengganti tag pembukanya lalu tag penutupnya . Namun jika kalian menggunakan extension ini maka kalian hanya perlu mengganti tag pembukanya saja, untuk tag penutupnya akan otomatis berubah sesuai tag yang kalian ganti.


3. Intellisense For Css Claassnames

  • Extension By : Zignd
  • Downloads : +4.049.763
  • Rating: 3.5 of 5

Extension ini akan menampilkan Intellisense pada codingan kalian, jadi kalian tinggal pilih saja code yang kalian inginkan.


4. Css Peek

  • Extension By : Pranay Prakash
  • Downloads : +2.453.087
  • Rating: 3.6 of 5

Jika kalian ingin melihat code pada elemen html kalian sudah di beri apa saja pada css maka kalian harus menggunakan extension ini.


5. Indent Rainbow

  • Extension By : Oderwat
  • Downloads : +2.625.776
  • Rating: 4.8 of 5

Pada Visual Studio Code akan terdapat garis untuk menhubungkan div pembuka dan div penutup yang akan mempermudah kita melihat elemen-elemen yang ada di dalam div tersebut, namun untuk membedakan div satu dengan div-div lainnya akan sulit karena semua adalah garis putih tipis.


Jika kalian menggunakan extension ini maka kalian dapat dengan mudah membedakan div satu dengan div-div lainnya karena setiap div akan terdapat sebuah garis yang selebar satu tab dengan warna yang berbaeda-beda.


6. Bracket Pair Colorrizer

  • Extension By : CoenaraadS
  • Downloads : +6.837.512
  • Rating: 4.6 of 5

Extension ini akan mempermudah kalian untuk melihat code yang ada di dalam kurung kurawal dengan warna yang berbeda-beda sehingga kalian dapat melihat bahwa code ini berada di kurung kurawal dari code parentnya.


7. Beautify

  • Extension By : HookyQR
  • Downloads : +7.732.589
  • Rating: 4.3 of 5

Extension ini akan membantu kalian merapikan code kalian yang berantakan, kalian cukup tekan kanan di keyboard lalu pilih format dokumen dan pilih beutify.


Sekian Artikel kali ini, semoga ini dapat kalian jadikan sebagai referensi kalian buat membatu mempercepat penyelesaian project kalian, sampai jumpa di artikel-artikel lainnya.
