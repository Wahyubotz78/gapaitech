Mengoneksikan jaringan Menggunakan Cisco Paket Tracer


Koneksikan jaringan dengan 3 Router dan 3 Pc menggunakan Cisco Paket Tracker !

Jawab:

1. Merancang tampilan


2. Melakukan konfigurasi alamat ip address dan gateway untuk komputer pertama


3. Melakukan konfigurasi alamat ip address dan gateway untuk komputer kedua


4.Melakukan konfigurasi alamat ip address dan gateway untuk komputer ketiga


5. Melakukan konfigurasi ip address pada FastEthernet0/0 untuk router pertama


6. Melakukan konfigurasi ip addess pada serial 2/0 untuk router pertama


7. Melakukan konfigurasi ip addess pada FastEthernet0/0 untuk router kedua


8. Melakukan konfigurasi ip addess pada serial 2/0 untuk router kedua


9. Melakukan konfigurasi ip addess pada serial 3/0 untuk router kedua


10. Melakukan konfigurasi ip addess pada serial 2/0 untuk router ketiga


11. Melakukan konfigurasi ip addess pada serial 3/0 untuk router ketiga


12. Melakukan RIP Routing untuk router pertama


13. Melakukan RIP Routing untuk router kedua


14. Melakukan RIP Routing untuk router ketiga


15. Nampak pada gambar setiap komputer saling bertukar pesan dan semuanya bersatatus Successfull
