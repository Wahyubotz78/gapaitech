Gampang, Begini Cara Memberi Warna Pada Gambar SVG



Lihat Tutorial Dalam Bentuk Video




Hai teman-teman apa kabar semua, saya harap kalien pada baik-baik saja, di kesempatan ini saya ingin memberi tutorial atau cara bagaimana mewarnai file gambar dalam format svg. Sebelum masuk ke tutorialnya kalien udah pada tau gak apa itu svg?


Pengertian Svg

Menurut wikipedia Scalable Vector Graphics (SVG) adalah format gambar yang menggunakan XML (Extensible Markup Language) sebagai dasar untuk membentuk gambar vektor dua dimensi. SVG adalah format gambar yang dikembangkan oleh World Wide Web Consortium (W3C) sejak tahun 1999.


Jadi svg itu gambar yang dibuat menggunakan bahasa markup yaitu xml (Extensible Markup Language) yang dikembangkan oleh world wide web atau yang biasa kita kenal www pada tahun 1999.


Mewarnai File Svg

Dalam dunia pemrograman baik itu pemrograman web, android atau perograman apa pun itu pasti membutuhkan yang namanya ui desain atau tampilan antarmuka pengguna, terutama desain pada button contohnya seperti gambar di bawah ini



Button pada gambar diatas adalah button yang dibuat dengan file gambar svg. Dengan memodifikasi warna dan ukurannya yang sesuai dengan keinginan.


Sebenarnya ada banyak cara yang dapat kita gunakan untuk mewarnai file gambar svg agar sesuai keinginan kita, pada tutorial ini saya akan memberikan cara bagaimana biasanya saya mewarnai file gambar dalam format svg. dan kita akan membuat button sepeti gamar di atas, namun disetiap buttonnya mempunyai warna yang berbeda agar lebih mudah memahami cara mengubah warna pada gambar svg tersebut.


Cara mewarnai Gambar Svg


Code HTML Sebelum Gambar di Warnai

index.html

<!DOCTYPE html>

<html lang="en">

<head>

<meta charset="UTF-8">

<meta http-equiv="X-UA-Compatible" content="IE=edge">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>cara memwarnai gambar svg</title>

<link rel="stylesheet" href="style.css">

</head>

<body>

<div class="container">

<ul>

<li><img src="img/facebook.svg" alt=""></li>

<li><img src="img/github.svg" alt=""></li>

<li><img src="img/instagram.svg" alt=""></li>

<li><img src="img/telegram.svg" alt=""></li>

</ul>

</div>

</body>

</html>


Code CSS Sebelum Gambar di Warnai

style.css

* {

margin: 0;

padding: 0;

}

.container {

max-width: 400px;

margin: 300px auto;

}

.container ul {

display: flex;

list-style: none;

}

.container li {

margin: 0 20px;

}

.container li img {

width: 30px;

height: 30px;

}


Hasil dari codingan di atas terlihat semua gambar masih dalam keadaan warna default



Code HTML Untuk Mewarnai Gambar SVG

index.html

<!DOCTYPE html>

<html lang="en">

<head>

<meta charset="UTF-8">

<meta http-equiv="X-UA-Compatible" content="IE=edge">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>cara memwarnai gambar svg</title>

<div class="container">

<li><img class="facebook" src="img/facebook.svg" alt=""></li>

<li><img class="github" src="img/github.svg" alt=""></li>

<li><img class="instagram" src="img/instagram.svg" alt=""></li>

<li><img class="telegram" src="img/telegram.svg" alt=""></li>

</ul>

</div>

</body>

</html>


Code CSS Untuk Mewarnai Gambar SVG

style.css

* {

margin: 0;

padding: 0;

}

.container {

max-width: 400px;

margin: 300px auto;

}

.container ul {

display: flex;

list-style: none;

}

.container li {

margin: 0 20px;

}

.container li img {

width: 30px;

height: 30px;

}

.facebook {

filter: invert(20%) sepia(100%) saturate(4345%) hue-rotate(241deg) brightness(101%) contrast(95%);

}

.github {

filter: invert(13%) sepia(7%) saturate(0%) hue-rotate(127deg) brightness(103%) contrast(96%);

}

.instagram {

filter: invert(46%) sepia(73%) saturate(1561%) hue-rotate(318deg) brightness(114%) contrast(101%);

}

.telegram {

filter: invert(44%) sepia(14%) saturate(5692%) hue-rotate(212deg) brightness(96%) contrast(88%);

}

.facebook:hover, .github:hover, .instagram:hover, .telegram:hover {

opacity: 0.6;

cursor: pointer;

}


Hasil dari codingan di atas terlihat semua gambar SVG sudah berwarna.

Untuk memberi warna pada gambar SVG kita tidak bisa menggunakan property color, background-color, ataupun background pada css, tetapi menggunakan property filter. namun untuk membuat sebuah warna yang kita inginkan dengan property filter sepertinya sulit, di sini saya menggunakan bantuan dari subuah CSS Filter Generator yang terdapat di website codepen.io.


Ayo kita bahas bagaimana cara menggunakan CSS Filter Generator tersebut

1. Tentukan warna apa yang ingin kalian gunakan lalu salin atau copy, ingat warna harus berdasarkan code hex , kalian bisa ke website htmlcolorcodes untuk melihat code hex setiap warna


2. Buka CSS Filter Generator pastekan code hex yang kalian salin tadi di box target color lalu tekan Compute Filters, seperti gambar di bawah ini


3. Terakhir salin property CSS filter yang ada di bawah, seperti gambar di bawah ini
